Grafana dan prometheus

instal grafana dan prometheus di server (centos 7) :

1. install grafana akan pakai port : 3000 \n 2. install prometheus akan pakai port : 9090 \n 3. pastikan sudah bisa diakses semua web nyaa. \n 4. install disisi client yang akan dimonitoring : node.exporter \n 5. cara install tinggal download file node.exporter sesuaikan dengan server kalau linux cari .deb atau .npm atau aapalah. kalau windows server cari .exe \n setelah install node.exporter di client atau server yang akan dimonitoring \n untuk menambahkan targets > kita masuk ke server prometheus lalu masuk folder /etc/prometheus > edit file prometheus.yml \n tambahkan ip target, interval ,dll \n setelah itu reload di browser pada menu status > targets \n untuk menampilkan grafik di grafana > \n login dari web ke grafana dashboard > menu setting > data source > add data source > input ip prometheus \n untuk membuat graph nya > menu dashboard > browse > + new folder atau new dashboard > masuk + panel \n
